Welcome to the xxxterm site.

XXXTerm is a minimalists web browser. It strives to be vi-like for heavy keyboard users. Development has just begun and currently it is very spartan.

Version numbers are based the cvs tag. For example, this is how V1.22 looks like: $xxxterm$

It is based on webkit using GTK+ Please refer to the man page for more details.

You can download the source via anoncvs at:
CVSROOT=anoncvs@anoncvs.freedaemon.com:/xxxterm

To subscribe to CVS changes send an email to majordomo@freedaemon.com and in the email body say: subscribe xxxterm-changes.
